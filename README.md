## Featured Projects

### Rentra — Open-Source Property Management Platform
> The property management platform built for independent landlords.

A full-stack, self-hostable platform for managing rental properties across multiple countries and currencies. Covers everything from tenant and lease management to financial tracking, document storage, and multi-factor authentication. Built with a clean separation between frontend and backend, containerized with Docker, and designed to be privacy-first with no vendor lock-in. Mobile support via CapacitorJS (iOS & Android) is in active development.

**Highlights:** multi-currency support, OAuth + 2FA + passkey auth via BetterAuth, secure signed-URL document storage, Drizzle ORM with PostgreSQL, Doppler secret management, and a full repository/service/controller architecture on the backend.

**Stack:** React 19 · TypeScript · Vite · Tailwind CSS · ShadCN · TanStack Query · Zustand · Node.js · Express 5 · PostgreSQL · Drizzle ORM · Supabase · BetterAuth · Zod · Docker · CapacitorJS

[![Rentra Repo](https://img.shields.io/badge/View_Repo-Rentra-181717?style=flat&logo=github)](https://github.com/Abdullah73k/Rentra)

### EthosAI — Agentic Blockchain Intelligence
> Agentic market intelligence and behavioral awareness for Ethereum wallets.

Analyzes on-chain wallet activity, correlates it with live market data, and produces explainable behavioral insights using an LLM. Includes a Discord voice coaching bot that joins your server, listens via speech-to-text, reasons with your wallet context loaded in, and responds through a synthesized voice in real time.

**Stack:** ElysiaJS · Bun · Prisma · PostgreSQL · React 19 · Tailwind v4 · ShadCN · DeepSeek LLM · Deepgram STT · ElevenLabs TTS · Etherscan API · CoinGecko

### EcoSim — AI Climate Policy Simulator *(Hackathon)*
> Interactive platform for simulating the environmental impact of policy decisions.

Lets users adjust policy variables (carbon tax, renewable adoption, deforestation) and instantly see modeled outcomes for temperature rise, CO2 concentration, and sea level change. Features a Three.js-powered 3D globe with live warming heatmaps and an AI advisor that explains consequences in plain language.

**Stack:** React 18 · TypeScript · React Three Fiber · Vite · FastAPI · DeepSeek LLM · TanStack Query

---

### Wallet Scam Detector ML — Blockchain Security
> ML pipeline for classifying cryptocurrency wallets as safe or malicious.

Trains an XGBoost classifier on behavioral features derived from transaction history — frequency, duration, token patterns — using Stratified K-Fold Cross-Validation and automated hyperparameter tuning via RandomizedSearchCV. Goes beyond blocklists by detecting suspicious wallets that haven't been reported yet.

**Stack:** Python · XGBoost · Scikit-Learn · Pandas · NumPy · Matplotlib · Etherscan API
